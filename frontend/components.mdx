## Hero & Landing Components

### Hero

Main hero section with animated headline and interactive rotor.

**Location**: `src/components/Hero.jsx`

**Key Features**:
- Staged animation sequence with useEffect
- Text decryption effect using custom `useTextDecrypt` hook
- Interactive rotor component
- Statistics section with motion animations
- Responsive grid layout

**Animation Stages**:

```jsx
const [stage, setStage] = useState(0);

useEffect(() => {
  const t1 = setTimeout(() => setStage(1), 300);   // Show rotor
  const t2 = setTimeout(() => setStage(2), 800);   // Show tag
  const t3 = setTimeout(() => setStage(3), 1200);  // Decrypt headline
  const t4 = setTimeout(() => setStage(4), 2500);  // Show description
  const t5 = setTimeout(() => setStage(5), 3000);  // Show CTAs
  const t6 = setTimeout(() => setStage(6), 3500);  // Show stats
  return () => [t1, t2, t3, t4, t5, t6].forEach(clearTimeout);
}, []);
```

**Text Decryption Effect**:

```jsx
const headline = t("hero.headline");
const { displayText } = useTextDecrypt(headline, stage >= 3);

```

### Services

Interactive service selector with rotating wheel interface.

**Location**: `src/components/Services.jsx`

**Key Features**:
- Circular rotor with 7 service icons
- Interactive selection with smooth rotation
- Motion-based animations
- Responsive sizing (mobile vs desktop)
- AnimatePresence for smooth transitions

**Service Configuration**:

```jsx
const SERVICES = [
  { id: "desarrollo-web", icon: <CodeIcon /> },
  { id: "servidores-cloud", icon: <ServerIcon /> },
  { id: "correo", icon: <MailIcon /> },
  { id: "ciberseguridad", icon: <ShieldIcon /> },
  { id: "automatizacion", icon: <AutomationIcon /> },
  { id: "agentes-ia", icon: <AIIcon /> },
  { id: "consultoria", icon: <ConsultingIcon /> },
];

const COUNT = SERVICES.length;
const STEP = 360 / COUNT; // 51.43 degrees per service
```

**Rotation Logic**:

```jsx
const ringRotation = useMotionValue(0);
const springRotation = useSpring(ringRotation, { damping: 25, stiffness: 150 });

function selectService(newIndex) {
  let delta = newIndex - prevActive.current;
  // Handle wraparound for shortest rotation path
  if (delta > COUNT / 2) delta -= COUNT;
  if (delta < -COUNT / 2) delta += COUNT;
  ringRotation.set(ringRotation.get() - delta * STEP);
  prevActive.current = newIndex;
  setActive(newIndex);
}
```

## Component Props Patterns

Most components follow these patterns:

<Note>
  **Translation Integration**: Components use `const { t } = useT()` to access translations
  
  **Motion Integration**: Animations use `motion` from `motion/react` with `useInView` hooks
  
  **Routing**: Internal links use `<LocaleLink>` instead of `<Link>` for language-aware routing
</Note>

## Custom Hooks Used

Components leverage these custom hooks:

- **useTextDecrypt**: Animated text reveal effect (Hero.jsx:53, Services.jsx:121)
- **use3DTilt**: 3D tilt effect on mouse move (Plans.jsx:17)
- **useCountUp**: Animated number counting
- **useStickyScroll**: Sticky scroll behavior

See the [Internationalization](/frontend/internationalization) page for details on the `useT` hook.
