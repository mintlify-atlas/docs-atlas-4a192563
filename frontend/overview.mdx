# Frontend Overview

The Enigma Developers frontend is a modern React application built with cutting-edge technologies for performance, internationalization, and exceptional user experience.

## Technology Stack

The application is built with the following core technologies:

  <Card title="React 19" icon="react">
    Latest React version with improved performance and concurrent features
  </Card>
  <Card title="Vite 6" icon="bolt">
    Lightning-fast build tool and development server
  </Card>
  <Card title="React Router 7" icon="route">
    Client-side routing with code splitting and lazy loading
  </Card>
  <Card title="Tailwind CSS 4" icon="paintbrush">
    Utility-first CSS framework with custom theme
  </Card>
  <Card title="Motion (Framer)" icon="sparkles">
    Powerful animation library for interactive UI
  </Card>
  <Card title="Multi-language" icon="globe">
    Full Spanish and English support with custom i18n system
  </Card>
</CardGroup>

## Application Entry Point

The application bootstraps in `main.jsx`:

```jsx
import { StrictMode } from "react";
import { createRoot } from "react-dom/client";
import { BrowserRouter } from "react-router-dom";
import App from "./App.jsx";
import "./index.css";

createRoot(document.getElementById("root")).render(
  <StrictMode>
    <BrowserRouter>
      <App />
    </BrowserRouter>
  </StrictMode>
);
```

## Core Architecture Patterns

### 1. Context-Based State Management

The application uses React Context for global state:

- **LanguageContext**: Manages current language, translations, and language switching
- No external state management library needed for this scale

### 2. Component Composition

Components follow a modular, composable pattern:

- Page components compose smaller feature components
- Shared UI components are highly reusable
- Custom hooks extract reusable logic

### 3. Route-Based Code Splitting

Pages are lazy-loaded to improve initial bundle size:

```jsx
const ServicesPage = lazy(() => import("./pages/ServicesPage.jsx"));
const PlansPage = lazy(() => import("./pages/PlansPage.jsx"));
const About = lazy(() => import("./pages/About.jsx"));
```

<Note>
  The Home page is NOT lazy-loaded since it's the primary landing page and should load immediately.
</Note>
