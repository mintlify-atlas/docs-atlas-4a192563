## Architecture

The Enigma Developers backend is a modern REST API built with Express 5, Prisma 6, and PostgreSQL 16. It handles contact forms, lead generation, complaint submissions, and provides email notifications.

## Application Setup

The main application file (`src/index.js`) configures middleware and routes:

```javascript src/index.js
import express from "express";
import cors from "cors";
import helmet from "helmet";
import rateLimit from "express-rate-limit";
import { prisma } from "./db.js";
import contactRoutes from "./routes/contact.js";
import leadsRoutes from "./routes/leads.js";
import complaintsRoutes from "./routes/complaints.js";

const app = express();
const PORT = process.env.PORT || 4000;

app.set("trust proxy", 1);
app.use(helmet());
```

## Security Middleware

### Helmet

Helmet sets secure HTTP headers to protect against common vulnerabilities:

```javascript src/index.js
app.use(helmet());
```

### CORS Configuration

Dynamic origin validation from environment variables:

```javascript src/index.js
const allowedOrigins = [
  process.env.CORS_ORIGIN || "http://localhost",
  (process.env.CORS_ORIGIN || "").replace("://", "://www."),
];

app.use(
  cors({
    origin(origin, cb) {
      if (!origin || allowedOrigins.includes(origin)) return cb(null, true);
      cb(null, false);
    },
    methods: ["GET", "POST"],
  })
);
```

### Rate Limiting

Protects API from abuse with 50 requests per 15-minute window:

```javascript src/index.js
const apiLimiter = rateLimit({
  windowMs: 15 * 60 * 1000,  // 15 minutes
  max: 50,                    // 50 requests per window
  standardHeaders: true,
  legacyHeaders: false,
  message: { error: "Demasiadas solicitudes, intenta de nuevo más tarde." },
});

app.use("/api", apiLimiter);
```

### JSON Payload Limit

Prevents large payload attacks:

```javascript src/index.js
app.use(express.json({ limit: "10kb" }));
```

<Note>
All API endpoints are protected by rate limiting and payload size restrictions.
</Note>

## Error Handling

Global error handling middleware:

```javascript src/index.js
// 404 handler
app.use((_req, res) => {
  res.status(404).json({ error: "Ruta no encontrada" });
});

// 500 handler
app.use((err, _req, res, _next) => {
  console.error(err.stack);
  res.status(500).json({ error: "Error interno del servidor" });
});
```

## Server Initialization

Graceful startup and shutdown with database connection:

```javascript src/index.js
async function start() {
  try {
    await prisma.$connect();
    console.log("Conectado a PostgreSQL");

    app.listen(PORT, () => {
      console.log(`Backend corriendo en puerto ${PORT}`);
    });
  } catch (error) {
    console.error("Error al iniciar el servidor:", error);
    process.exit(1);
  }
}

process.on("SIGTERM", async () => {
  await prisma.$disconnect();
  process.exit(0);
});

start();
```
